# 🚀 Product Manager Portfolio & Career Quiz

Лендинг-портфолио для профессии **Product Manager** с интерактивным тестом на профориентацию. Проект демонстрирует навыки верстки, работы с Vanilla JS и внимание к деталям UI/UX.

## 📋 О проекте

Это не просто сайт-визитка, а полноценный инструмент для знакомства с профессией. Сайт адаптирован под темную тему, содержит актуальные данные по зарплатам, roadmap развития и FAQ. Отдельная страница с квизом помогает пользователю определить, подходит ли ему роль продакта.

### ✨ Ключевые особенности
- **Адаптивный дизайн:** Корректное отображение на десктопах, планшетах и мобильных устройствах.
- **Интерактивный квиз:** 5 вопросов с логикой подсчета баллов и тремя вариантами результатов (Vanilla JS).
- **Чистый код:** Семантическая верстка, CSS-переменные для легкой кастомизации, отсутствие лишних зависимостей.
- **Актуальный контент:** Реалистичные грейды зарплат (Junior/Middle/Senior), хард/софт скиллы и пошаговый план входа в профессию.

##  Стек технологий

| Технология | Назначение |
| :--- | :--- |
| **HTML5** | Семантическая разметка, доступность |
| **CSS3** | Flexbox/Grid, анимации, переменные, медиа-запросы |
| **JavaScript (ES6+)** | Логика квиза, DOM-манипуляции, обработка событий |
| **Google Fonts** | Шрифт Inter для современной типографики |
